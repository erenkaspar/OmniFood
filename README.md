# 🥗 Omnifood - AI-Powered Food Subscription

This project is a modern, real-world website for a fictional food subscription service, developed as the final project of Jonas Schmedtmann's "Build Responsive Real-World Websites with HTML and CSS" course.

## 🚀 About the Project

Omnifood is a tech-first company focused on consumer well-being through an AI-centric approach to healthy dieting. The website is designed to handle complex layouts, responsive transitions, and professional-grade UI/UX patterns.

## 🛠️ Tech Stack

- **HTML5:** Utilized for semantic structure and content organization.
- **CSS3:** Built using advanced layouts, including Flexbox and CSS Grid.
- **JavaScript:** Implemented for mobile navigation toggling and dynamic data like the current year.
- **Ionicons:** Integrated for high-quality, scalable vector icons.
- **Google Fonts:** Uses the "Rubik" typeface for a modern look.

## 🌟 Key Features

- **Advanced Typography & Color System:** Follows a strict design system with defined font sizes, weights, and primary/tint/shade color palettes.
- **Fully Responsive Design:** Includes a comprehensive queries.css file with specific breakpoints for desktops, tablets, and mobile phones (e.g., 1344px, 1200px, 944px, 704px, and 544px).
- **Reusable Components:** Features a modular system for buttons, grids, and typography (e.g., .container, .grid--2-cols, .btn--full) to ensure code maintainability.
- **Interactive Elements:** Features smooth scrolling, hover effects, and a custom-built mobile navigation menu.
- **Optimized Hero & CTA Sections:** High-impact sections designed for user conversion and engagement.

## 📂 Project Structure

```text
├── css/
│   ├── style.css    # Main styling for all sections
│   ├── general.css  # Global styles and reusable design system
│   └── queries.css  # Media queries for responsiveness
├── js/
│   └── script.js    # Mobile menu logic and copyright year automation
├── img/             # Optimized images for meals, logos, and gallery
└── index.html       # Semantic HTML structure
```
